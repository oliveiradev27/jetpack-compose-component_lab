# Jetpack Compose Component Lab

Repositório dedicado à exploração e criação de componentes utilizando Jetpack Compose com o objetivo de fixar conceitos, testar abordagens de UI declarativa e aprofundar o entendimento da toolkit moderna de UI do Android.

## 🎯 Objetivo

Este projeto funciona como um laboratório prático de aprendizagem.
A ideia é construir pequenos componentes isolados para entender melhor:

funcionamento do Jetpack Compose

composição de layouts

gerenciamento de state

recomposição

criação de componentes reutilizáveis

boas práticas de UI declarativa no Android

Em vez de apenas consumir exemplos prontos, este repositório busca aprender construindo.

## 🧪 Estrutura do Repositório

Cada componente é desenvolvido de forma modular e independente, permitindo experimentação e análise isolada.

Exemplos de experimentos que podem aparecer no repositório:

- Botões customizados
- Cards
- Inputs
- Layouts compostos
- Estados derivados
- Uso de remember
- Uso de derivedStateOf
- Gerenciamento de recomposição
- Componentes reutilizáveis

A ideia é que cada experimento represente um pequeno estudo focado em um conceito específico.

## 🧠 Conceitos Explorados

Durante o desenvolvimento dos componentes, alguns conceitos importantes do Compose são investigados:

- UI declarativa
- State hoisting
- Recomposição
- Performance em Compose
- Design de componentes reutilizáveis
- Separação entre UI state e business state

## 🚀 Motivação

Aprender frameworks de UI apenas lendo documentação raramente é suficiente.

Este repositório nasce da ideia de que:

A melhor forma de aprender um framework é construindo pequenas peças e entendendo profundamente como elas funcionam.

Cada componente aqui representa um pequeno experimento de aprendizado.
